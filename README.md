## 🏗️ Architettura

Il progetto è diviso in due repository principali:

### 🎨 Frontend

👉 https://github.com/portanovadomenico-dot/localproFE

* Interfaccia utente
* Gestione stato
* Comunicazione con API

---

### ⚙️ Backend

👉 https://github.com/portanovadomenico-dot/localproBE

* API REST
* Database e logica server
* Autenticazione e sicurezza

---

## 🛠️ Tecnologie utilizzate

### Frontend

* ⚛️ Angular
* 🎨 CSS / Html / Bootstrap
* 🔗 Fetch API

### Backend

* 🟢 Node.js / Java 
* 🚀 Express / Spring 
* 🗄️ Database  MySQL 

---

## 🔗 Come funziona

1. Il frontend invia richieste al backend tramite API
2. Il backend gestisce la logica e comunica con il database
3. I dati vengono restituiti al frontend e mostrati all’utente
